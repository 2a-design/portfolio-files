# portfolio-files

File hosting for assets used by external sites.

## Video hosting

Store video files in `/videos`.

Example URL pattern for a file named `intro.mp4`:

- `https://raw.githubusercontent.com/2a-design/portfolio-files/main/videos/intro.mp4`

For better browser delivery via CDN, you can also use:

- `https://cdn.jsdelivr.net/gh/2a-design/portfolio-files@main/videos/intro.mp4`
